🚀 راهنمای ساخت APK از ARKA PREMIUM

⚠️ توجه: چون Android Studio نصب نیست، از روش آنلاین استفاده می‌کنیم.

مرحله 1: آپلود فایل‌ها

فایل‌های پروژه باید روی یک سرور HTTPS قرار بگیرند. می‌توانید از یکی از این روش‌ها استفاده کنید:

گزینه A: GitHub Pages (رایگان)

  1. یک repository در GitHub بسازید
  2. فایل‌های پوشه www را آپلود کنید
  3. از Settings → Pages فعال کنید

گزینه B: Netlify Drop (سریع‌ترین)

باز کردن Netlify Drop

فقط پوشه www را drag & drop کنید!

مرحله 2: ساخت APK با PWABuilder

رفتن به PWABuilder
  1. URL سایت آپلود شده را وارد کنید
  2. روی "Start" کلیک کنید
  3. صبر کنید تا آنالیز کامل شود
  4. "Package for stores" را انتخاب کنید
  5. "Android" را انتخاب کنید
  6. تنظیمات:
    • Package ID: com.arka.premium
    • App name: ARKA PREMIUM
  7. "Download" را کلیک کنید

مرحله 3: دریافت APK

فایل ZIP دانلود می‌شود که شامل:

  • app-release-signed.apk - فایل آماده نصب
  • app-release-bundle.aab - برای Google Play Store
